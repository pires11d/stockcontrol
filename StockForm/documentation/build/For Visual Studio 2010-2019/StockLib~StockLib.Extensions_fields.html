ToDimension Dictionary containing all SI units and their corresponding MLT values  
Units Unit conversion dictionary. Uses the following equation:

Y = A + B.X, where:

Y = SI value

X = Unit value

Ex.: Converting 60 degrees celsius (°C) to Kelvin (K)

Y = 273.15 + 1 * 60.0

Y = 333.15 K
